In order to use an icon as a part of usa-icon it must be selected in the module which declares the component where you wish to display the icon.

To select an icon you wish you use, you must first import the icon from either ngx-bootstrap-icons or @gsa-sam/ngx-uswds-icons into the module which declares your component.

import { NgxBootstrapIconsModule, square } from 'ngx-bootstrap-icons';
import { add, IconModule } from '@gsa-sam/ngx-uswds-icons';

Next you must add NgxBootstrapIconsModule to your imports and call it's pick function. As an input to this function you should provide an object which contains all of the icons you imported in the previous step

@NgModule({
 imports: [
  CommonModule,
  IconModule,
  NgxBootstrapIconsModule.pick({add, square})
 ],
 declarations: [YourComponent],
 exports: [YourComponent]
})

Finally in your template you can include usa-icon and set the value of the icon input to be one of the icons you just selected.

<usa-icon [icon]="'add'" [size]="'2x'"></usa-icon>
